int main ( )

Entry point for the load-balancer simulation.

Requests the starting server count and simulation duration, creates processing and streaming load balancers, generates the initial queue, and executes both load balancers for the selected number of cycles.

Returns
0 when the simulation completes successfully, or a nonzero value when the user enters invalid input.
